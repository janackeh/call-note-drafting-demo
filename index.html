A human-in-the-loop support workflow demo. Pick a scenario, toggle the prompt iteration, draft a CRM note, then edit it and see exactly what you changed.

Rule-based portfolio demo — not connected to live customer systems. Nothing you type is sent anywhere or stored.

This is a draft. Review and edit before posting — AI can be wrong.

What this demonstrates

  • Prompt iteration — toggle between full-summary (v1) and result-only (v2) output.
  • Structured CRM output — a clean Subject line plus a 1–3 sentence Call Summary.
  • PII awareness — emails, phones, and card-like numbers are flagged and masked before drafting.
  • Human edit & review — the draft is editable; a diff shows exactly what the human changed.
  • Outcome-focused v2 — the current prompt restricts the note to the call result, not every action taken.
